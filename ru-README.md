# Анализ магазина натуральной косметики

## 1. Описание

 Проект для анализа данных магазина натуральной косметики, исследования характеристик продуктов представленных в магазине, поведения клиентов и моделей продаж. Используются: SQL, Python, и Power BI. Анализ выделяет ключевые тенденции, самые продаваемые категории продуков и т.д.. Прокт включает в себя очистку данных, разведочный анализ данных (EDA), дашборд, и структурированную документацию для демонтрации практических навыков анализа данных в контексте розничной торговли и косметики.

> ## DISCLAIMER

>Проект основан на вымышленном магазине натуральной косметики. Все данные сгенерированы рандомно при помощи сервиса Mockaroo.com для учебных целей и составления портфолио. Магазин, покупатели, заказы, и т.д. не представляют никакую реальную компанию или рельных личностей.

 ## 6. Как использовать/открывать

 ### 6.1 SQL запросы: 

 SQL запросы написаны для PostgreSQL.  

Чтоб запустить их:

1. Создайте базу данных cosmetics в PostgreSQL.

2. Импортируйте CSV файлы из папки _`tables/`_.

3. Откройте файлы _`.sql`_ в pgAdmin Query.

4. Запустите запросы (f5)

### 6.2 Python 

Анализ Python расположен в папке _`notebooks/`_.  

**Чтобы запустить**:

1) Установите Python версии 3.10+

2) Запустите программу Jupiter Notebook

3) Откройте файлы _`.ipnyb`_  (etl и eda)

4) Убедитесь, что  CSV файлы из папки  _`tables/`_ находятся по тому же пути, чтобы код запускался корректно.

### 6.3 Power BI

Дашборд расположен в папке  _`powerbi`_.

1) Скачайте файл _`.pbix`_. 

2) Откройте его в программе Power BI Desktop 

## 7. Результаты

Таблицы:
- customers.csv (*покупатели*) состоит из 4 столбцов (идентификатор клиента, имя клиента, город, пол) и 325 строк
- products.csv (*продукты*) состоит из 5 столбцов (идентификатор продукта, название продукта, бренд, категория, цена) и 115 строк
- sales.csv (*продажи*) состоит из 8 столбцов (идентификатор продажи, дата, идентификатор продукта, количество, идентификатор клиента, скидка (Да/Нет), метод оплаты, процент скидки) 5000 строк

### 7.1 Диаграмы Python: 
- тенденция ежемесячного дохода (monthly revenue trend) 
- самые продаваемые продукты (best selling products)
- цена и количество проданной продукции (price vs. quantity sold)
- величина скидки и доход (discount rate vs. revenue)
- средний доход по размеру скидки (avg revenue by discount)
- доход и тенденция скидок (revenue & discount  trends)
- тепловая карта корреляций (correlation heatmap)
- кластеризация клиентов (customers clustering)
- топ бренды (top brands)

**Выводы:**
- Диаграммы не показывают четкой взаимосвязи между ценой на продукт (price) и количеством проданного продукта (quantity). Клиенты магазины постоянно покупают 1-3 единицы продукта, независимо от цены.
- Также нет взаимосвязи между величиной скидки и полученным доходом. Скидки составляют 1-10%, и в пределах этого диапазона размеры дохода широко разбросаны.
- Тепловая карта не показывает корреляции между большинством показателей.
  
  **Это связано с тем, что данные для проекта были сгенерированы рандомно**

### 7.2 Диаграммы Power BI:

- Доход по гендеру (Gender based revenue)
- Доход по способам оплаты (Revenue by payment method)
- Доход по месяцам (Revenue by month)
- Доход по брендам (Revenue by brand)
- Доход по категории продуктов (Revenue by category)
- Топ 5 брендов по городам (Top 5 brands by city)
- Количество покупателей по гендеру (Customers by gender)
- Количество продаж по способу оплаты (Sales by payment method)
- Количество продаж по месяцам (Sales by month)
- Количество продаж по брендам (Sales by brand)
- Количество продаж по категориям (Sales by category)

**Выводы:**
- Женщины потратили 180 000 евро, мужчины - 152 000 евро, что на ~15,6% меньше, чем женщины.
- По методу оплаты: Онлайн оплатили 117 000 евро, что составляет примерно 35,2% от всего дохода, по карте оплатили примерно 111 000 евро (примерно 33,4%), наличными оплатили 105 000 евро (примерно 31,4%)
- Месяцы с наиболее высоким доходом: май и октябрь (промерно по 30 000 евро), и с самым низкимдоходом: сентябрь (примерно 25 000 евро)
- Категория с самым высоким доходом: "Сыворотки, масла для лица и целевые средства" (Serums, face oils & targeted treatments) - примерно 27 000 евро, с самым низким доходом: "Уход за губами и кожей вокруг глаз" (Eye & lip Care/Specialized areas)
- Бренд с самым высоким доходом: Vintner's Daughter (41 000 евро), и с самым низким доходом: Barulab (469 евро)
- Количество покупок по месяцам: наибольшее количество покупок было совершено в январе (439), а наименьшее в апреле (390)
- Количество продаж по категориям: наибольшее количество единиц продукции было продано в категории "Маски, экзофолианты и уход для лица" (Masks,exofoliants & face treatment), наименьшее количество продукиции в категории "Уход за волосами" (Hair care)
- Количество проданных единиц по брендами: наибольшее количесвтво единиц (332) у бренда The Ordinary, и наименьшее количество (30) у бренда Camille Rose
